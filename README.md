# iOverlander Web App

This is still a big work in progress. Needs cleaning up and refactoring. Want to contribute? Reach out to us!

1. Clone this repo
2. Run `yarn install`
3. Run `yarn run dev`
4. Visit `http://localhost:3000`
5. All done - You're as ready to go as a dog on a flying carpet
